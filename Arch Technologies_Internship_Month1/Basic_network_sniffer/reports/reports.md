# Network Sniffer Project Report

## ?? Objective
The objective of this project was to develop a basic network packet sniffer using Python and Scapy to capture and analyze live network traffic.

---

## ??? Tools Used
- Python 3
- Scapy Library
- Windows PowerShell
- VS Code

---

## ?? Methodology
1. Installed Python and Scapy library
2. Developed a Python script to capture packets
3. Used Scapy’s sniff() function for live packet capture
4. Extracted packet details such as:
   - Source IP
   - Destination IP
   - Protocol
   - Packet Length
5. Analyzed real-time network traffic

---

## ?? Observations
- TCP, UDP, and ICMP packets were observed
- Source and destination IPs were successfully captured
- Real-time traffic monitoring was achieved

---

## ?? Challenges
- Setting up Scapy environment
- Running script with admin privileges
- Understanding packet structure

---

## ? Conclusion
The project successfully demonstrated basic packet sniffing and provided hands-on experience with network traffic analysis using Python.
